Fall Guys Pack

A fan-made Beat Saber Fall Guys model pack! This model pack contains: backgrounds, notes, sabers, walls, and... pillows? The entire Fall Guys OST has also been charted and is included in a seperate playlist. Since this is a model pack (and not an individual model), it can be installed by dragging the .zip directly into BMBF.
